# RideX - Car Rental Web Application

RideX is a car rental web application that allows users to rent cars for their transportation needs. It provides a user-friendly interface for browsing and renting cars, as well as managing rented cars.

## Features

- User registration and authentication
- Browse available cars
- Rent cars based on user preferences
- View and manage rented cars
- Return rented cars
- Search for cars by location and availability

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Setup and Installation

1. Clone the repository or download the project files.
2. Import the database schema by executing the SQL dump file located at `database/ridex.sql`.
3. Configure the database connection settings in the `config.php` file.
4. Upload the project files to your web server or run a local server environment (e.g., XAMPP, WAMP, or MAMP).
5. Access the application through your web browser.

## File Structure

- `index.php`: The main landing page of the application.
- `login.php` and `register.php`: Pages for user authentication and registration.
- `renting_car.php`: The main page after user authentication, showing available cars for rent.
- `rent_car.php`: Page for renting a car, where users can select the desired car and rental details.
- `rented_cars.php`: Page for viewing and managing rented cars by the user.
- `remove_car.php`: Backend script for removing a rented car from the database.
- `test.php`: Configuration file for the database connection.
- `styles.css`: CSS styles for the application.
- `scripts.js`: JavaScript code for client-side functionality.
